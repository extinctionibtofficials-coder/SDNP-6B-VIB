Firestore Rules Belum Diatur (Akses Ditolak)

Aplikasi mendeteksi error "Missing or insufficient permissions" dari Firebase. Ini berarti aturan keamanan (Security Rules) Firestore Anda masih terkunci.
Silakan buka Firebase Console → Firestore Database → Rules, lalu ganti rules Anda dengan aturan berikut agar aplikasi bisa membaca & menulis data: 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
